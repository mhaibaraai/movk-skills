# 报告输出格式

按分析深度与是否涉及多机构对比三选一。表格中未从原文获取到的字段填「未提及」，不要臆造。**预测性数字必须注明是机构预测还是已发生事实，不得混淆**（如"IEA 预测 2030 年石油需求见顶"与"2025 年全球石油需求同比增长 X%"是两类不同性质的表述）。

六个分析维度（各格式据此展开）：

1. **核心数据摘录** — 产量/储量/价格/需求预测等具体数字，注明单位、同比/环比、数据口径与来源页码
2. **关键结论与观点** — 机构原文的判断性表述，区分"机构预测/观点"与"已发生事实"
3. **市场趋势与技术方向判断** — 供需平衡变化、价格走势判断、能源转型/新能源替代节奏、炼化一体化等结构性趋势
4. **投资与项目动态** — 资本开支计划、新项目投产/最终投资决定（FID）、并购、产能扩张或退出
5. **风险与不确定性** — 地缘政治、监管/碳政策、价格波动，以及机构自身声明的前瞻性表述免责条款
6. **产业链与关联行业影响** — 按上游（勘探开采）→中游（炼化）→下游（化工品/新材料）分层，延伸到关联行业（新能源车、化纤、化肥等）

## 格式 A：单机构/单主题深度分析报告

用于重点报告（如 IEA WEO、企业年报）或用户明确要求全面解读时。

```markdown
# {机构} 《{报告名称}》洞察摘要

## 一句话概要

[最精炼地概括报告核心结论]

## 报告基本信息

| 项目 | 内容 |
|------|------|
| 机构 | ... |
| 报告类型 | 年度报告 / 可持续发展报告 / 月度报告 / 研究报告 |
| 发布日期 | ... |
| 覆盖周期 | ... |
| 语言 | 中文 / 英文 |

## 核心数据摘录

| 指标 | 数值 | 同比/环比 | 单位 | 来源（章节/页码） |
|------|------|-----------|------|--------------------|
| ... | ... | ... | ... | ... |

## 关键结论与观点

- {观点一}（机构预测 / 已发生事实）
- {观点二}（机构预测 / 已发生事实）

## 市场趋势与技术方向判断

...

## 投资与项目动态

...

## 风险与不确定性

...

## 产业链影响评估

- 上游：...
- 中游：...
- 下游：...

## 关联报告

- [报告名称](链接) — 关联说明

## 信息来源

- [标题](链接) — official_site / WebSearch 补充，发布日期
```

## 格式 B：行业动态要点速览

用于快速查询或单条新闻/公告的速览。

```markdown
# {主题} 行业动态速览

机构：{机构} | 报告类型：{类型} | 发布日期：{日期}

核心数据：

- {数据点 1}
- {数据点 2}

关键观点：

- {观点 1}（机构预测 / 已发生事实）
- {观点 2}

风险提示：{一句话}

来源：[链接] — official_site / WebSearch 补充
```

## 格式 C：多机构观点对比

旗舰场景，用于多个机构对同一议题存在分歧判断时（如 IEA vs OPEC 对全球石油需求峰值年份的分歧）。

```markdown
# {主题} 多机构观点对比

| 维度 | {机构 1} | {机构 2} | {机构 3} |
|------|----------|----------|----------|
| 核心预测数字 | ... | ... | ... |
| 关键假设 | ... | ... | ... |
| 与其他机构的分歧点 | ... | ... | ... |

分歧原因分析：...

综合判断建议：...

信息来源：逐条列出链接与来源层级（official_site / WebSearch 补充）
```
